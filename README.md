# Texas Hold'Em Simulaton App
This App functions as a simulation Hold'Em app. The user inputs a desired # of AI opponents, starting chips, and blind amounts. They then play hands of poker against the AI, who are coded to have random personalities (aggressive, normal, passive, each of which alter the action (ex. aggresive players have an auto +.15 aggression, which can alter the action to make them raise)) and evaluate their hand.

# Features:
Inputs for Opponents, Chips, Blinds
Auto shuffling and dealer movement
Table and player display with card graphics
Toggle for show/hide opponent hands
AI decision tree system
