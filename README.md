# App = Poker Hand Simulator
# This app allows a user to simulate a hand of poker. They can choose to reveal their opponents hand before the game starts (maybe to help someone understand the game) or play the more traditional way of only seeing their two cards. AI players will act out and then community cards will be dealt, just as a normal poker game

# Name: Joshua Lord

# Intended Users: people that enjoy card games

# Planned Features:
#  Input a certain number of AI players (whose playstyle is randomly determined)
#  Full 52-card deck that auto shuffles
#  Simulate real poker interactions (bet, check, fold, community cards)
# Hand evaluator that determines which player has the best hand
